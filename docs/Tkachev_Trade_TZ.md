## 1. ОПИСАНИЕ ПРОЕКТА

### 1.1 Общая информация
**Tkachev Trade** — мобильное приложение для анализа и торговли криптовалютами, разработанное на основе искусственного интеллекта. Приложение предназначено для крипто-трейдеров всех уровней и объединяет в себе инструменты для анализа рынка, идентификации торговых возможностей и выполнения сделок.

### 1.2 Назначение
Приложение служит комплексным решением для:
- Анализа движения цен и волатильности криптовалют
- Выявления потенциальных торговых сигналов (пампы, позиции ликвидации)
- Мониторинга рынка в режиме реального времени
- Быстрого доступа к торговым площадкам
- Получения рекомендаций от AI-ассистента

### 1.3 Целевая аудитория
- Начинающие крипто-трейдеры (новички)
- Опытные трейдеры (профессионалы)
- Инвесторы в криптовалюты
- Члены школы криптовалют Дмитрия Ткачева
- Пользователи, заинтересованные в пассивном анализе рынка

## 2. ФУНКЦИОНАЛЬНЫЕ ТРЕБОВАНИЯ

### 2.1 Основные модули приложения

#### 2.1.1 МОДУЛЬ АНАЛИЗА ПАМПОВ (Pump Detection Module)
**Описание:** Инструмент для выявления резких скачков цены монет (пампы)

**Функциональность:**
- Мониторинг движений цены в режиме реального времени
- Автоматическое выявление пампов по заданным параметрам:
  - Процент роста цены за определённый период
  - Объём торговли
  - Скорость приращения цены
- Уведомления при обнаружении пампа
- История пампов с архивом данных
- Статистика пампов (частота, амплитуда, длительность)
- Фильтры по:
  - Размеру пампа (малые, средние, крупные)
  - Временному диапазону
  - Монетам/парам
  - Биржам

**Данные для отслеживания:**
- Binance (основная биржа)
- Bybit
- OKX
- Mexc
- Другие крупные криптовалютные биржи

#### 2.1.2 МОДУЛЬ ЛИКВИДАЦИЙ (Liquidation Map)
**Описание:** Карта ликвидаций — визуализация позиций, находящихся под угрозой ликвидации на фьючерсных рынках

**Функциональность:**
- Отображение уровней ликвидации для основных монет (BTC, ETH и др.)
- Визуальная карта скопления лонг и шорт позиций
- Оценка объёма ликвидаций в USD
- Уведомления при приближении к уровням массовой ликвидации
- Исторические данные по ликвидациям
- Фильтры по:
  - Типу позиций (лонги/шорты)
  - Размеру ликвидации
  - Монетам

**Источники данных:**
- Binance Futures
- Bybit
- OKX Futures
- Deribit

#### 2.1.3 МОДУЛЬ СТАКАНОВ (Order Book / Market Depth)
**Описание:** Отображение онлайн заявок на покупку и продажу (Ask/Bid) в режиме реального времени

**Функциональность:**
- Визуализация стакана с двумя колонками (Buy/Sell)
- Цветовое обозначение объёмов
- Отслеживание крупных ордеров (киты)
- Мониторинг скорости изменения стакана
- История изменения стакана за последний период
- Настройка глубины стакана (показываемые уровни)
- Анализ уровней поддержки и сопротивления на основе стакана
- Интеграция с торговыми парами
- Фильтры по:
  - Размеру ордеров
  - Волатильности
  - Монетам

#### 2.1.4 МОДУЛЬ GPT АНАЛИЗА (AI Analysis Module)
**Описание:** Встроенный чат с AI-ассистентом на основе GPT для анализа монет

**Функциональность:**
- Диалоговый интерфейс с GPT
- Анализ криптовалют по запросу пользователя
- Формирование краткого анализа по монете включая:
  - Техническую позицию
  - Фундаментальные показатели
  - Рекомендации
  - Потенциальные риски
- История диалогов
- Сохранение интересующих анализов
- Возможность задать свой вопрос про анализ
- Поддержка команд типа:
  - "Анализ BTC"
  - "Почему упала цена ETH?"
  - "Прогноз по SOL на 24 часа"
  - "Какие монеты в тренде сейчас?"

#### 2.1.5 МОДУЛЬ ИНДИКАТОРА TKACHEV AI SIGNAL
**Описание:** Интеграция с индикатором TradingView на основе AI

**Функциональность:**
- Подключение к индикатору Tkachev AI Signal на TradingView
- Получение торговых сигналов (Buy/Sell) в мобильное приложение
- Отслеживание срабатывания триггеров (6 триггеров подряд = сигнал)
- Автоматические Take Profit и Stop Loss
- График доходности в реальном времени
- Расчёт потенциального заработка на основе депозита
- Статистика выполнения сигналов
- Фильтрация сигналов по:
  - Временным интервалам (5 min, 15 min, 1h)
  - Монетам
  - Типам сигналов (лонг/шорт)
  - Уровню риска

#### 2.1.6 МОДУЛЬ ГРАФИКОВ (Trading Charts)
**Описание:** Встроенные интерактивные графики для анализа цен

**Функциональность:**
- Свечные графики (candlestick charts)
- Выбор временных периодов (1m, 5m, 15m, 1h, 4h, 1d, 1w)
- Наложение технических индикаторов:
  - SMA, EMA, WMA
  - RSI, MACD, Bollinger Bands
  - Stochastic, ADX, CCI
- Рисование линий и фигур (trendlines, support/resistance)
- Сохранение анализов
- Экспорт скриншотов
- История просмотров графиков

#### 2.1.7 МОДУЛЬ УВЕДОМЛЕНИЙ И ОПОВЕЩЕНИЙ
**Описание:** Система push-уведомлений о критических событиях на рынке

**Функциональность:**
- Уведомления о пампах
- Уведомления о ликвидациях
- Уведомления по срабатыванию торговых сигналов
- Уведомления о достижении заданных уровней цены
- Уведомления о высокой волатильности
- Возможность настройки типов уведомлений
- Контроль частоты уведомлений
- Звуковые и вибрационные оповещения

## 4. АРХИТЕКТУРА ПРИЛОЖЕНИЯ

### 4.1 Стек технологий

#### Frontend (Мобильное приложение)
- **iOS:** Swift или React Native
- **Android:** Kotlin или React Native
- **Framework:** React Native (универсальное решение) или native для каждой платформы
- **State Management:** Redux / Mobx / Riverpod
- **UI Framework:** Flutter Material / iOS HIG
- **Графики:** Lightweight Charts / TradingView Lightweight Charts
- **Реал-тайм:** WebSocket (для push-данных)

#### Backend
- **API:** Node.js (Express) / Python (FastAPI) / Go (для высокой производительности)
- **Database:** PostgreSQL (основные данные), Redis (кэширование), MongoDB (логи, аналитика)
- **Message Queue:** RabbitMQ / Apache Kafka (для потока событий)
- **Real-time:** WebSocket через Socket.io или прямые WS соединения
- **AI/ML:** OpenAI API, Anthropic Claude, или локальные LLM модели
- **Контейнеризация:** Docker, Kubernetes

#### Интеграции
- **Биржи API:** 
  - Binance API (REST + WebSocket)
  - Bybit API
  - OKX API
  - CoinMarketCap API
  - CoinGecko API
- **TradingView:** Pine Script API для индикаторов
- **AI:** OpenAI GPT-4 / Claude API
- **Платежи:** Stripe / PayPal

## 14. ГЛОССАРИЙ ТЕРМИНОВ

- **Памп (Pump)** - резкое увеличение цены актива
- **Ликвидация** - закрытие позиции при падении цены ниже допустимого уровня
- **Стакан (Order Book)** - список всех активных ордеров на покупку и продажу
- **Лонг/Шорт** - открытие позиции на рост/падение цены
- **Take Profit (TP)** - уровень прибыли, при котором закрывается выигрышная позиция
- **Stop Loss (SL)** - уровень убытков, при котором закрывается убыточная позиция
- **Smart Money** - крупные торговцы/инвесторы, чьи движения анализируются
- **Спот-торговля** - покупка/продажа криптовалют без использования леверидж
- **Фьючерсы** - контракты на поставку актива в будущем с использованием леверидж
- **Леверидж** - использование заёмных средств для увеличения позиции
- **Волатильность** - интенсивность колебаний цены
- **Индикатор** - математический инструмент для анализа цен
- **MAU** - Monthly Active Users (активные пользователи в месяц)
- **MRR** - Monthly Recurring Revenue (ежемесячно повторяющийся доход)
- **NPS** - Net Promoter Score (индекс лояльности)
