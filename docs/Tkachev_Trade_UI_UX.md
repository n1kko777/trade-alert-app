## 2. ПОЛЬЗОВАТЕЛЬСКИЕ СЦЕНАРИИ

### Сценарий 1: Новичок ищет торговые сигналы

**Persona:** Алекс, 25 лет, новичок в крипто
**Задача:** Получить торговый сигнал для быстрого заработка
**Временной интервал:** 5 минут

**Шаги:**
1. Открывает приложение → видит Dashboard с портфелем
2. Переходит на вкладку "Мои сигналы"
3. Видит 3 активных сигнала Tkachev AI Signal (Pro)
4. Выбирает сигнал "ETHUSDT BUY" с TP $2500, SL $2400
5. Нажимает "Открыть позицию" → переходит на Binance через в-приложение браузер
6. Выставляет сделку согласно рекомендациям
7. Возвращается в приложение, видит трекинг сделки в real-time
8. Получает push-уведомление при достижении TP
9. Закрывает сделку и видит прибыль +$245

**Ключевые моменты:**
- Простота доступа к сигналам
- Интеграция с биржами для быстрого открытия позиций
- Real-time мониторинг

---

### Сценарий 2: Опытный трейдер анализирует рынок

**Persona:** Иван, 35 лет, профессиональный трейдер
**Задача:** Провести глубокий анализ нескольких монет перед торговым днём
**Временной интервал:** 30 минут

**Шаги:**
1. Открывает приложение, видит "Горячие монеты" на Dashboard
2. Переходит на "Анализ пампов" → видит 5 монет с ростом >100%
3. Выбирает одну монету, видит:
   - График с индикаторами (SMA, EMA, RSI)
   - Стакан (Order Book)
   - Карту ликвидаций
4. Запускает GPT анализ с вопросом "Потенциал монеты PEPE?"
5. Получает развёрнутый анализ с рекомендациями
6. Смотрит уровни ликвидации, видит риск на уровне $0.000087
7. Открывает 4 разных монеты на одном экране (сплит-вид)
8. Сравнивает их поведение, определяет лучшую для торговли
9. Выставляет стоп-лосс и тейк-профит согласно ликвидационной карте
10. Добавляет монеты в "Избранное" для быстрого доступа

**Ключевые моменты:**
- Сложные инструменты анализа
- Быстрый доступ к нескольким данным одновременно
- Интеграция разных модулей в единый workflow

### Сценарий 3: Инвестор отслеживает портфель

**Persona:** Мария, 40 лет, долгосрочный инвестор
**Задача:** Проверить статус инвестиций в портфеле
**Временной интервал:** 10 минут (ежедневно)

**Шаги:**
1. Открывает приложение
2. Видит на Dashboard:
   - Текущая стоимость портфеля: $5,234.56
   - Доходность сегодня: +12.34%
   - Доходность месяца: +45.67%
   - Доходность всё время: +234%
3. Открывает раздел "Портфель"
4. Видит распределение активов (pie chart):
   - BTC 40%
   - ETH 35%
   - SOL 15%
   - Другое 10%
5. Кликает на BTC, видит детали:
   - Цена при покупке: $35,000
   - Текущая цена: $42,534
   - Количество: 0.125 BTC
   - Прибыль: +$945 (+21.5%)
6. Проверяет каждый актив
7. Видит рекомендацию GPT: "Пересбалансировка рекомендуется"
8. Нажимает "Пересбалансировать", получает план действий
9. Закрывает приложение, остаётся спокоен

**Ключевые моменты:**
- Быстрый общий взгляд на портфель
- Детальная информация по каждому активу
- Минимум кликов для проверки

### Сценарий 4: Обучающийся трейдер учится через приложение

**Persona:** Сергей, 22 года, студент, новичок в крипто
**Задача:** Пройти обучение и сделать первую успешную сделку
**Временной интервал:** 1-2 часа

**Шаги:**
1. Открывает приложение, видит "Школа" на нижней навигации
2. Выбирает курс "Крипто от А до Я"
3. Начинает с модуля "Основы" (10 видеоуроков по 5-10 минут)
4. Смотрит видео о:
   - Что такое криптовалюта
   - Как работают биржи
   - Спот vs фьючерсы
5. После каждого видео проходит мини-тест (3 вопроса)
6. Переходит на модуль "Технический анализ"
7. Видит живые примеры на графике ETH прямо в приложении
8. Нажимает "Практика", попадает в режим демо-торговли
9. Открывает первую позицию на виртуальные $1,000
10. Отслеживает позицию, видит как она развивается
11. Закрывает с небольшой прибылью, получает сертификат
12. Видит свой прогресс в "Профиле": 45% курса пройдено

**Ключевые моменты:**
- Интерактивное обучение
- Встроенные примеры
- Демо-торговля для практики
- Прогресс и мотивация

## 5. СОДЕРЖАНИЕ В РЕАЛЬНОМ ВРЕМЕНИ (Real-time Data)

### 5.1 Обновления данных по умолчанию

| Функция | Частота обновления | Канал |
|---------|-------------------|-------|
| Цены | Каждую 1 сек | WebSocket |
| Стакан | Каждые 500 мс | WebSocket |
| Пампы | Каждые 5 сек | WebSocket |
| Ликвидации | Каждые 10 сек | WebSocket |
| Портфель | Каждую 1 сек | REST API |
| Новости | Каждые 60 сек | REST API |
| AI сигналы | Мгновенно | WebSocket + Push |
| Графики | Каждую 1 сек | WebSocket |
